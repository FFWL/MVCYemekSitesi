# MVCYemekSitesi
.NET MVC5 kullanarak oluşturulan admin panelli tarif sitesi

-Admin Paneline ulaşmak için /Admin/Index urlsini kullanabilirsiniz. Örnek olması açısından kullanıcıadı:fenrir şifre:123
adlı kayıttan admin paneline ulaşım sağlayabilirsiniz
-İlişkili tablolar için aktif/pasif işlemleri kullanılmıştır. Fakat sizlerin görmesi açısından silme işlemi sadece aktif
kayıtları pasif yapmaktadır. View tarafında görüntülenme durumunda tüm kayıtlar gözükmektedir.
-ilişkili olmayan Recipe tablosundaki değerler direkt olarak silinebilmektedir.
